## Usages

Add following lines to your buildscripts.

```groovy
buildscript {
    ext {
        outside_activity_version = '0.0.1'
    }
}
```

```groovy
repositories {
    maven {
        url 'http://dl.bintray.com/s64/maven'
    }
}

dependencies {
    compile "jp.s64.android:outsideactivity:${outside_acivity_version}"
}
```
